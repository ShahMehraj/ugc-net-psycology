# UGC NET Psychology — PYQ Practice & Mock Test

A static web app to attempt UGC NET Psychology previous-year papers in the official
NTA style — **Practice mode** (instant answer + detailed explanation) and a timed
**CBT mock test** with a detailed scorecard.

The application lives in [`app/`](./app). See [`app/README.md`](./app/README.md) for
features, how to run it locally, and how it deploys to GitHub Pages.

## Quick start

```bash
cd app
npm install
npm run dev
```

## Currently available

- **June 2023** — full 150-question paper (Part-1 General Aptitude + Psychology), with
  explanations.

## Note on source material

Question content is transcribed from published previous-year papers and a third-party
PYQ book **for educational practice only**. The raw copyrighted PDFs/books are **not**
included in this repository. Answer keys are cross-checked but may contain errors —
always verify against the official UGC/NTA keys.
